Agente de IA para WooCommerce — conecta WhatsApp, Instagram y web chat en minutos.

== Description ==

ORQO conecta tu tienda WooCommerce con un agente de IA que atiende a tus clientes
en WhatsApp, Instagram y el chat web de tu sitio, 24/7.

= Características =

* Chat web embebible con un snippet automático
* Agente de IA configurado para tu negocio (nombre, personalidad, idioma)
* Conexión con WhatsApp e Instagram desde el mismo dashboard
* Sin código — instalación en 2 pasos
* Compatible con WooCommerce y cualquier tema WordPress

= Instalación =

1. Sube el plugin a /wp-content/plugins/orqo-chat/
2. Actívalo desde el panel de Plugins
3. Ve a Ajustes → ORQO Chat
4. Pega tu API Key (la encuentras en dashboard.orqo.io → Configuración → Motor de Agentes)
5. Guarda y listo — el widget aparece automáticamente en tu tienda
